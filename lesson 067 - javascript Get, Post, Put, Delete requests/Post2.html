Comparison:         .then() vs                          await
Feature	.then()	await
Works with Promises?	    ✅ Yes	                        ✅ Yes
Looks synchronous?	    ❌ No	                        ✅ Yes
Can chain multiple steps?   ✅ Yes (via .then().then())	✅ Yes (just write normally)
Inside async function?	    ❌ Not required	                ✅ Required
Returns a Promise?	    ✅ Yes	                        ❌ No — returns final value
